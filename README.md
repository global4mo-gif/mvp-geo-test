# MVP Geo Test

Минималистичный сайт-презентация для объяснения, почему проект является не классическим geo-split, а тестовым запуском стартапа с предварительной маркетинговой упаковкой.

## Что внутри

- интерактивная структура работ;
- калькулятор бюджета 290-300 тысяч рублей;
- распределение рекламного бюджета по трём локациям;
- демо-сценарии страниц: лендинг, предзаказ, отчёт;
- отдельный проектный документ в PMI-логике для клиента Алены;
- всплывающие подсказки и модальные окна.

## Локальный запуск

Откройте `index.html` в браузере или запустите простой сервер:

```bash
python3 -m http.server 5173
```

## GitHub Pages

Проект рассчитан на публикацию как статический сайт через GitHub Pages из ветки `main`, папка `/`.
